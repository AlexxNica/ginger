Installing Ginger

Ginger depends on Ginger-base. First step is to download and install it.

Then, download a Ginger package according to your operating system.

We provide packages for the latest version of Fedora and Ubuntu. Those packages were built on and for x86_64 machines.

After downloading a package using the default packaging management tool in your system. You can do it by command line like below:

Fedora:

sudo yum install <kimchi.rpm>

Ubuntu:

sudo dpkg -i <kimchi.deb>

openSUSE:

sudo zypper install <ginger-base.rpm>

Restarting wokd service

After installing Ginger, you just need to restart wokd service to get it up:

sudo systemctl restart wokd.service

or:

sudo service wokd restart

Note: The above command line may vary depends on your operating system.

Then access WoK in your favorite browser https://<machine-ip>/8001

Have fun! =)
